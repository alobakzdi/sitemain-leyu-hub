# sitemain-leyu-hub

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
仓库本身不针对任何特定域名、网站或服务，仅作为静态 HTML 资料的集中存放与版本管理。

## 项目简介

`sitemain-leyu-hub` 旨在提供一个简洁、稳定的 HTML 页面归档空间。  
每个页面都是独立的，彼此之间没有依赖关系。  
你可以在此仓库中浏览、引用或归档各类 HTML 文档。

## 目录说明

```
.
├── index.html          # 仓库首页 / 页面索引
├── pages/              # 存放独立 HTML 页面
│   ├── example-page-1.html
│   ├── example-page-2.html
│   └── ...
├── assets/             # 公共资源（如 CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：主要归档目录，每个 HTML 文件为一个独立页面。
- `assets/`：存放页面所需的公共资源文件，便于复用和维护。

## 页面归档说明

- 所有页面均为纯 HTML，不依赖外部数据库或后端服务。
- 页面内容、样式、脚本均打包在对应文件或 `assets/` 目录中。
- 归档的页面可能包含不同的主题或用途，彼此独立，互不干扰。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面。
- 提交时请确保页面文件完整、资源引用正确。
- 不建议在页面中嵌入外部不可控资源，以免影响长期可用性。
- 欢迎通过 Issue 或 Pull Request 提出改进建议，但请注意保持仓库定位一致。

## 许可

本仓库内容仅用于归档与展示目的。  
除非页面内另有声明，否则默认不授予任何商业使用或再分发权限。
